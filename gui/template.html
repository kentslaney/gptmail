Welcome to GPTMail's web interface!
GPTMail is a system that trains a text machine learning model (GPT-2, a type of Transformer) on someone's emails, and autocompletes a prompt based on what it thinks that person would say. You can find more info on its GitHub.
The main part of this page is just a textbox for the prompt and autocompletion results. The text in bold is the prompt, the part you can edit, and once you autocomplete the prompt, the results will show in normal weight text. An example prompt has been filled in to get you started.

Click on the "autocomplete" button to start running the machine learning model. This may take a while, and the button will display a loading animation while the server is working. Alter the number in the textbox to the right to change how many words the server writes.
If there's more than one machine learning model, you can change which one the server autocompletes with by selecting a different option here.
